Host nodejmc #peut etre n’importe quel nom Hostname 172.105.4.22 #le serveur qu’on veut acceder Port 22 User root

le public key doit etre mis dans le serveur qu’on veut acceder il doit etre dans le fichier (a creer s’ils n’existent pas) .ssh/authorized_keys la meme chose en utilisant ssh-copy-id -i ~/.ssh/id_rsa.pub root@172.105.7.26 id_rsa.pub - le public key qu’on veut copier 172.105.7.26 - le serveur qu’on veut y acceder

ssh-i ~/.ssh/privateKey root@172.105.7.22 ssh agent - it’s to save the key in memory to don’t set each time you connect to server

to find ssh-agent ps aux |grep ssh-agent

to activate it eval “$(ssh-agent)”

to add key to ssh-agent ssh-add ~/thePathToTheKeyThatWeWantToAdd only the first time he will ask you for the passphrase (unless u don’t close the terminal)
