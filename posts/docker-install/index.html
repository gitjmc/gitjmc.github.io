Posted 2022-02-27 Updated 2025-04-20 1 min read

Cet article montre comment peut-on installer docker dans Deepin1 version 20.x et plus.

install Docker

Prerequistes

2
3
4

$ sudo apt-get update
$ sudo apt install -y apt-transport-https ca-certificates gnupg2 curl software-properties-common
$ sudo curl -fsSL https://download.docker.com/linux/debian/gpg | sudo apt-key add -
$ sudo apt-key fingerprint 0EBFCD88

Check source.list file

path /etc/apt/sources.list path /etc/apt/sources.list.d/docker-ce.list

1

$ cat /etc/apt/sources.list

Install docker

1
2
3

$ sudo apt-get update
$ sudo apt-get install -y docker-ce
$ docker --version

install docker-compose

Source pour installer docker compose3.

1
2
3

$ sudo curl -L "https://github.com/docker/compose/releases/download/1.29.2/docker-compose-$(uname -s)-$(uname -m)" -o /usr/local/bin/docker-compose
$ sudo chmod +x /usr/local/bin/docker-compose
$ docker-compose version
