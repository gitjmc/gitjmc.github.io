List of 100 essential Vim commands

By Jerbi Mohamed Chamseddine
Posted 2025-02-06 Updated 2025-02-22 5 min read

Vim is a powerful text editor, but it has a steep learning curve. Below is a list of 100 essential Vim commands categorized for easy reference. These commands will help you navigate, edit, and manage files efficiently in Vim.

Basic Navigation

  1. h - Move cursor left.
  2. j - Move cursor down.
  3. k - Move cursor up.
  4. l - Move cursor right.
  5. w - Move to the start of the next word.
  6. b - Move to the start of the previous word.
  7. e - Move to the end of the current word.
  8. 0 - Move to the beginning of the line.
  9. $ - Move to the end of the line.
  10. gg - Move to the first line of the file.
  11. G - Move to the last line of the file.
  12. :n - Move to line number n (e.g., :10 goes to line 10).
  13. Ctrl + f - Scroll forward one page.
  14. Ctrl + b - Scroll backward one page.
  15. Ctrl + d - Scroll down half a page.
  16. Ctrl + u - Scroll up half a page.
  17. H - Move to the top of the screen.
  18. M - Move to the middle of the screen.
  19. L - Move to the bottom of the screen.
  20. % - Jump to the matching bracket ({[]}).

Insert Mode

  1. i - Insert before the cursor.
  2. I - Insert at the beginning of the line.
  3. a - Insert after the cursor.
  4. A - Insert at the end of the line.
  5. o - Insert a new line below the current line.
  6. O - Insert a new line above the current line.
  7. s - Delete the character under the cursor and enter insert mode.
  8. S - Delete the entire line and enter insert mode.
  9. Esc - Exit insert mode and return to normal mode.

Editing Text

  1. x - Delete the character under the cursor.
  2. X - Delete the character before the cursor.
  3. dd - Delete the current line.
  4. dw - Delete from the cursor to the start of the next word.
  5. d$ - Delete from the cursor to the end of the line.
  6. d0 - Delete from the cursor to the beginning of the line.
  7. yy - Yank (copy) the current line.
  8. yw - Yank from the cursor to the start of the next word.
  9. y$ - Yank from the cursor to the end of the line.
  10. p - Paste after the cursor.
  11. P - Paste before the cursor.
  12. u - Undo the last action.
  13. Ctrl + r - Redo the last undone action.
  14. . - Repeat the last command.
  15. J - Join the current line with the next line.
  16. r - Replace the character under the cursor.
  17. R - Enter replace mode (overwrite text).
  18. ~ - Toggle case of the character under the cursor.
  19. >> - Indent the current line.
  20. << - Unindent the current line.
  21. == - Auto-indent the current line.

Search and Replace

  1. /pattern - Search forward for pattern.
  2. ?pattern - Search backward for pattern.
  3. n - Repeat the last search forward.
  4. N - Repeat the last search backward.
  5. :%s/old/new/g - Replace all occurrences of old with new in the entire file.
  6. :%s/old/new/gc - Replace with confirmation.
  7. :s/old/new/g - Replace in the current line.
  8. :s/old/new/gc - Replace in the current line with confirmation.
  9. * - Search for the word under the cursor.
  10. # - Search backward for the word under the cursor.

File Operations

  1. :w - Save the file.
  2. :wq - Save and quit.
  3. :q - Quit (if no changes were made).
  4. :q! - Quit without saving changes.
  5. :w filename - Save the file with a new name.
  6. :e filename - Open another file in the current buffer.
  7. :r filename - Insert the contents of another file into the current file.
  8. :saveas filename - Save the file with a new name and switch to it.
  9. :bn - Switch to the next buffer.
  10. :bp - Switch to the previous buffer.
  11. :bd - Close the current buffer.
  12. :ls - List all open buffers.
  13. :e! - Revert to the last saved version of the file.

Visual Mode

  1. v - Enter visual mode (character-wise).
  2. V - Enter visual line mode.
  3. Ctrl + v - Enter visual block mode.
  4. y - Yank (copy) the selected text.
  5. d - Delete the selected text.
  6. > - Indent the selected text.
  7. < - Unindent the selected text.
  8. ~ - Toggle case of the selected text.

Advanced Editing

  1. ciw - Change the inner word (delete and enter insert mode).
  2. c$ - Change from the cursor to the end of the line.
  3. c0 - Change from the cursor to the beginning of the line.
  4. ct<char> - Change until the specified character.
  5. cf<char> - Change up to and including the specified character.
  6. diw - Delete the inner word.
  7. dt<char> - Delete until the specified character.
  8. df<char> - Delete up to and including the specified character.
  9. gUw - Uppercase the current word.
  10. guw - Lowercase the current word.
  11. g~~ - Toggle case of the entire line.
  12. gqq - Format the current line to a specific width.
  13. :set number - Show line numbers.
  14. :set nonumber - Hide line numbers.
  15. :set wrap - Enable line wrapping.
  16. :set nowrap - Disable line wrapping.
  17. :set paste - Enable paste mode (prevents auto-indentation).
  18. :set nopaste - Disable paste mode.
  19. :set spell - Enable spell checking.

Tips for Using Vim Efficiently

  • Practice regularly to build muscle memory.
  • Use .vimrc to customize Vim settings and create shortcuts.
  • Learn to use macros (q) for repetitive tasks.
  • Explore plugins like NERDTree, vim-airline, and fzf for enhanced functionality.
