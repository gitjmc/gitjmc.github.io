Install nodejs on Deepin 20.x

By Jerbi Mohamed Chamseddine
Posted 2022-03-20 Updated 2025-02-22 1 min read

Cet article montre comment installer node1, npm et angular dans Deepin version 20 et plus.

Disons que Ça pas mal fonctionné!

Solution

Solution (tested on ssd-500)

Look for debian’s version-releases used in deepin.

buster - est la version de Debian de Deepin 20.4

If curl is not installed then you have to do

1
2

sudo apt update && sudo apt upgrade
sudo apt install curl
  1. Uninstall old node version
1

sudo apt -y remove nodejs
  1. Add the NodeSource package signing key
1
2
3
4
5

KEYRING=/usr/share/keyrings/nodesource.gpg
curl -fsSL https://deb.nodesource.com/gpgkey/nodesource.gpg.key | gpg --dearmor | sudo tee "$KEYRING" >/dev/null
# wget can also be used:
# wget --quiet -O - https://deb.nodesource.com/gpgkey/nodesource.gpg.key | gpg --dearmor | sudo tee "$KEYRING" >/dev/null
gpg --no-default-keyring --keyring "$KEYRING" --list-keys
  1. Add the desired NodeSource repository
1
2
3
4
5
6
7
8

# Replace with the branch of Node.js or io.js you want to install: node_6.x, node_8.x, etc...
VERSION=node_16.x
# Replace with the keyring above, if different
KEYRING=/usr/share/keyrings/nodesource.gpg
# The below command will set this correctly, but if lsb_release isn't available, you can set it manually:
DISTRO="$(buster)"
echo "deb [signed-by=$KEYRING] https://deb.nodesource.com/$VERSION $DISTRO main" | sudo tee /etc/apt/sources.list.d/nodesource.list
echo "deb-src [signed-by=$KEYRING] https://deb.nodesource.com/$VERSION $DISTRO main" | sudo tee -a /etc/apt/sources.list.d/nodesource.list
  1. update package lists and install Node.js
1
2

sudo apt-get update
sudo apt-get install nodejs
  1. chaeck installed version
1
2

node --version
npm --version

For more details take a look to this link

install angular cli

1

sudo npm install -g @angular/cli

1

ng --version
